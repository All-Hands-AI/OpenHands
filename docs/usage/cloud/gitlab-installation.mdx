## Prerequisites

- A GitLab account
- Access to OpenHands Cloud

## Installation Steps

1. Log in to [OpenHands Cloud](https://app.all-hands.dev)
2. If you haven't connected your GitLab account yet:
   - Click on `Log in with GitLab`
   - Authorize the OpenHands application
